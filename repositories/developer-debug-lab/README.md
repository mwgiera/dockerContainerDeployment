# Developer Debug Lab

Hands-on debugging scenarios that mirror common developer-platform support tickets.

## Scenarios
- Node.js runtime logic bug (`reduce` without an initial value)
- Python API parsing failure (`KeyError`)
- Missing dependency
- Missing environment variable

## Structure
- `node-api/` runtime bug and fixed version
- `python-service/` bug and fixed version
- `environment-issues/` configuration/dependency issues
- `logs/` sample failures
- `troubleshooting/` workflow and incident analysis
